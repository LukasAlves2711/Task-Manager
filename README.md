🎲 Running the Back End (server)


```bash
# Clone the repository
$ git clone <git@github.com:LukasAlves2711/Task-Manager.git>

# Install dependencies
$ npm install

# Run the application

$ node app.js

# The server will start on port:3000 - go to <http://localhost:3000>

```
